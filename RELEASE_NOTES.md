# EzraOS v2.1.0 Alpha

First public alpha release of EzraOS, an offline AI development environment built for Android through Termux.

## Features

- Offline AI chat powered by llama.cpp
- Custom EzraOS terminal interface
- Persistent user name with first-time setup
- Name can be changed in Settings
- Mode-specific greetings and focused assistants
- General Chat mode
- Java Programming mode
- Python Programming mode
- SQL Learning mode
- Git and GitHub mode
- Linux and Termux mode
- Bible Study mode
- Streaming AI responses
- Separate conversation history per mode
- Automatic history trimming to reduce context errors
- Notes module
- Project repository scanner
- System information
- Background llama-server management
- Works locally after the model is downloaded

## Requirements

- Android
- Termux
- llama.cpp Termux package
- Python
- curl
- git
- Around 1.2 GB free storage for the recommended model
- At least 4 GB physical RAM recommended

## Current Status

This is an alpha release. Bugs and compatibility issues may still occur.

EzraOS is an OS-like environment inside Termux. It does not replace Android.
